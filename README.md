# Description
This is a markdown slides file designed for in-class presentation.

# Installation 
You need Obsidian and `Advanced Slides` plugin to view this.

1. Install obsidian
2. Go to settings -> community plugins
3. Install `Advanced Slides` plugin, then enable it
4. Create/open an obsidian vault
5. put these markdown files in the vault
6. open the markdown file in obsidian
7. click the three-dot button on top right corner
8. click `start presentation`
