# General vs Spesifik

### General 

- LLM bisa memproses dan menghasilkan jenis data baru<!-- element class="fragment"-->
- LLM bisa dikatakan menjadi lebih "general" atau "generic" karena bisa dipakai oleh lebih banyak orang<!-- element class="fragment"-->
- Gemini 3 adalah model berbentuk multimodal<!-- element class="fragment"-->
- dampaknya luas, namun dangkal<!-- element class="fragment"-->

---
### Spesifik

- LLM akan beroperasi di bidang yang sangat amat narrow/spesifik<!-- element class="fragment"-->
- contohnya AlphaGo, AlphaFold, StockfishChess<!-- element class="fragment"-->
- hanya expert yang bisa menggunakan alat ini<!-- element class="fragment"-->
- dampaknya sangat sempit, tapi potensi impact lebih besar<!-- element class="fragment"-->
